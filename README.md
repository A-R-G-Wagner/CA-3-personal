# Alex Wagner PersonalProject
<br>
Back-end indeholder netbeans projektet
utils => SetupTestUsers indeholder startup data til databasen(brugere, roller, farver og relations)
Entity klasserne 'user' og 'role' er skaber grundlag for login funktionalitet.
DataFetcherFacade facaden bruges til at hente url'er ved hjælp af HttpUtils

<br>
Front-end indeholder react appen.
Hvis der er fejl under kørslen skal der muligvis installeres følgende:
 - npm install react-router-dom
 - npm install bootstrap
